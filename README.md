This project is the frontend (UI) of a Pharmacy Management System. It allows users to manage medicine inventory, track sales, and monitor stock with a clean and responsive interface. The UI is built using modern React practices and communicates with a .NET backend API.

🚀 Features
➕ Add new medicines with complete details
✏️ Edit existing medicine information
❌ Delete medicines from inventory
🔍 Search medicines by name or brand
📊 View sales records and total revenue
⚠️ Smart alerts for:
Low stock medicines
Expiring medicines
🛠️ Tech Stack
React 19 (Frontend library)
Vite (Fast build tool)
Axios (API communication)
CSS (Custom responsive styling)
